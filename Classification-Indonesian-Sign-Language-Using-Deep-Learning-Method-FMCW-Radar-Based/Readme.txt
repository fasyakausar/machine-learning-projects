Langkah-Langkah Pengambilan Dataset :

1. Dataset diambil dengan jarak kurang lebih 1,2 meter (120 cm) dari radar ke objek. 
2. Tinggi tripod dan radar adalah sepanjang kurang lebih 1 meter dari permukaan tanah.
3. Pergerakan bahasa SIBI adalah pergerakan huruf dimana jari bergerak namun lengan diam (contoh pergerakan di buku TA Oplib).
4. Dataset direkam dengan durasi 180 detik, 50 sampel, bandwith 240 MHz, Mode 2 uRAD Radar, MTI OFF.
5. Dataset direkam dengan duduk menyamping kiri dari radar dan posisi tangan ke tengah radar (contoh gambar di buku TA Oplib).
